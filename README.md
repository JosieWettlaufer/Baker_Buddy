# Baker_Buddy
Repository for the Baker Buddy Web Application using MERN stack

#Startup Tips 
1. MAKE SURE THERE ARE NO OTHER PROJECTS RUNNING ON THE SAME PORT!!!!
2. Running DB -> open compass, connect to BakerBuddyDB
3. Running Node -> use CLI, navigate to react-mongo-api, use <node /server.js>
4. Running React -> Use CLI, navigate to react-mongo-api/client, use <yarn /start>
5. Temporary URL -> localhost:3000/api/recipepages
